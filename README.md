## Instalando o projeto
O projeto depende da versão 17.0.10 do JDK. Com o JDK instalado:

1 - Acessar o diretório raiz:
```bash
cd sp-blog-api\
```
2 - Buildar o projeto:
```bash
.\gradlew.bat build
```
2 - Rodar o projeto:
```bash
java -jar .\build\libs\blogApi-0.0.1-SNAPSHOT.jar
